## New Basemap Upgrades Are Live in GSP.NET

We have released a major basemap update to improve clarity, speed, and day-to-day survey usability in the WebMap.

### What is new

- Added **OpenStreetMap HOT** for stronger road/building detail in many developing areas.
- Added **Carto Positron**, **Carto Voyager**, and **Carto Dark Matter** for clean, professional cartographic styles.
- Added **Esri World Topo Map** for dependable global topographic context.
- Added **Esri World Terrain + Reference** for terrain-focused viewing with readable labels.

### What changed

- The previous experimental Maxar-like imagery blend mode has been removed.
- Basemap order has been redesigned to prioritize practical field workflow and faster decision-making.
- Default map startup remains on **Esri World Imagery** for immediate high-detail viewing.

### New recommended basemap flow

1. Start with **Esri World Imagery** for visual ground detail.
2. Switch to **Esri World Topo Map** or **Esri World Terrain + Reference** for terrain interpretation.
3. Use **Carto Voyager/Positron** when overlay readability is the priority.
4. Use **OSM HOT** for urban and road-network context.

### Why this matters

These additions give surveyors and GIS users better context switching between:
- high-detail imagery,
- terrain/topographic understanding,
- and clean thematic basemaps for overlays and reporting.
